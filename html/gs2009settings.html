Redirecting

  
Enabling redirecting from YouTube / All search results to yt2009 and Wayback Machine.
 



Wayback Machine Date:  

Format must be like: '20100324182056' (default value), 'YYYYMMDDHHmmss'
(Y = year, M = month, D = day, H = 24h hour, m = minute, s = seconds)

yt2009 Instance Address:  

Format must be like: '192.168.0.X' '192.168.10.X:PORT' 'yt2009.example.com' 'localhost'
Please do not include http:// (same with https://, ftp://(?)) in address, or don't work.

If you want to redirect into yt2009, You must put your yt2009 instance address in above,
or redirecting to yt2009 will be disabled /or only with Wayback Machine.
